# What is this?

The repository is the source for https://douglangille.ca

It's coded using the [!Jekyll framework](https://jekyllrb.com/) and the source is managed on GitHub via the [!GitHub Pages](https://pages.github.com). 
The site is built and published using Netlify.

[![Netlify Status](https://api.netlify.com/api/v1/badges/2363cf5e-f6a0-4c9a-9755-3ae6bed26642/deploy-status)](https://app.netlify.com/sites/douglangille/deploys)

The base theme is [!Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes) and hacked by Yours Truly.
